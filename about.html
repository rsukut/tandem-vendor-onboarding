About the Vendor Onboarding Tool

Home

This tool is designed for use exclusively by Tandem employees. It streamlines the vendor onboarding process and ensures that all relevant teams receive the necessary information.

The source of truth for onboarding data is an Excel spreadsheet that is backed up daily to ensure data integrity. If there is any missing or incorrect information, updates can be made by the administrator.

The website, Microsoft Form, and backend process will be updated weekly, or immediately if there is an issue preventing use.

When you select "Begin Onboarding" and complete the Microsoft Form, the tool will send forms to:

  • IT
  • DevEx
  • Rena (for hardware orders)
  • Quality and HR (for notifications)
  • The tool can be updated as needed by the administrator to ensure all onboarding requirements are met.

    • For questions or issues, contact the administrator directly.
